# LABYRINTH 

Recursivity exercise made with [ReactJs](https://react.dev/)

## How to run

### Install

```bash
npm install
```

### Run
```bash
npm run start
```

### How it looks

The app give you a forest and you will need create a maze as you wish and the algorithm will find a path to solve the maze, if it exist.
